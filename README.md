# RunPika

It is a GNOME Shell Extension, where the speed of the Pikachu running in your taskbar indicates your CPU usage.

## Installation

>[!TIP]
> You need to have `Linux` + `GNOME Desktop` for this extension.

[RunPika | GNOME Shell Extensions Website](https://extensions.gnome.org/extension/9833/runpika)

## Thanks

A re-stylized derivative of [RunCat](https://github.com/win0err/gnome-runcat).
